##  Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Aroobadilshad-dev333/Arooba-Portfolio.git
cd Arooba-Portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```



### Animation Customization
All animations are CSS-based and can be modified in `src/index.css`:
- Typing animation speed
- Blob movement patterns
- Hover effect timing
- Transition durations
